.. _pygambit:


``pygambit`` Python package
===========================

Gambit provides a Python package, ``pygambit``, which provides access to
Gambit's features.  ``pygambit`` is available on PyPI
(https://pypi.org/project/pygambit/), and can be installed via ``pip``.
